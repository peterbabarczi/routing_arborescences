# Perfect Routing Arborescences for Fast Reroute

Program code and input topologies for finding perfect routing arborescences in Eulerian digraphs, published in:

Péter Babarczi and János Tapolcai, Perfect Routing Arborescences for Fast Reroute, in Proceedings of the IEEE International Conference on Computer Communications (INFOCOM), pp. 1-10, Tokyo, Japan, 2026.

# Usage
Run main.py with a given input topology from net/ to calculate routing arborescences. Results will be saved in res/.
